
# express + sequelize 생성 및 사용법


# config/config.json
db 접속정보 수정 
``` 
"development": {
    "username": 사용자명,
    "password": 설정한 비밀번호,
    "database": 접속할 데이터베이스명,
    "host": url,
    "dialect": "mysql"
  }
```


# 프로젝트 시작
``` 
npm start
``` 

# api 동작 확인 - 브라우저에 복붙!
``` 
http://localhost:3001/users 
```
